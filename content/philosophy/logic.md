# Logic & Argument

An **argument** is a set of statements where some (the **premises**) are offered as reasons to accept another (the **conclusion**). Logic studies which patterns of reasoning preserve truth.

## Validity vs. Soundness

- An argument is **valid** if the conclusion *must* be true whenever all premises are true. Validity is about structure, not whether the premises are actually true.
- An argument is **sound** if it is valid *and* all its premises are actually true. Soundness is the goal of good reasoning.

Example of a valid (and sound) argument:
1. All humans are mortal. (premise)
2. Socrates is human. (premise)
3. Therefore, Socrates is mortal. (conclusion)

A valid but unsound argument can have a false premise: "All birds can fly; a penguin is a bird; therefore a penguin can fly." The form is valid, but premise 1 is false.

## Deduction, Induction, Abduction

- **Deductive** reasoning aims for conclusions that follow with certainty from the premises.
- **Inductive** reasoning generalizes from observations; conclusions are probable, not certain (e.g., "the sun has risen every day, so it will rise tomorrow").
- **Abductive** reasoning infers the best explanation for the evidence (common in science and diagnosis).

## Common Fallacies

- **Ad hominem**: attacking the person instead of the argument.
- **Straw man**: misrepresenting a view to make it easier to attack. The antidote is *steelmanning* — stating the strongest version of an opposing view before responding.
- **Affirming the consequent**: "If P then Q; Q; therefore P" — invalid.
- **Begging the question**: assuming the conclusion within a premise.
- **False dilemma**: presenting only two options when more exist.

## Why It Matters

Clear definitions and explicit argument structure let us locate exactly where a disagreement lives: is a premise false, or is the inference invalid? This discipline underlies every other area of philosophy.
